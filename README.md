# 飛馬先進科技官方網站

Pegasus Advance Technology Ltd. 官方靜態網站（獨立官網，不依賴外部部落格）。

## 本機預覽

直接開啟 `index.html`，或在此資料夾執行：

```bash
python -m http.server 8080
```

瀏覽：`http://localhost:8080`

## 頁面

| 檔案 | 說明 |
|------|------|
| `index.html` | 首頁（實機照片輪播） |
| `about.html` | 關於我們 |
| `services.html` | 產品與解決方案（依公司簡報） |
| `gallery.html` | 實績相簿（可點擊放大） |
| `contact.html` | 聯絡我們 |
| `images/` | 官網照片素材 |

## 部署到 GitHub Pages

1. 將 `website` 內容推到 GitHub Repository
2. Settings → Pages → Source：`main` / `/ (root)`
3. 網址：`https://<帳號>.github.io/<repo>/`
